Welcome

This is an Ionic Kitchen Sink application featuring most of the components of the Ionic Framework and the Cordova Plugins.

View Code

You can view the source code for the given component by viewing clicking the button on the top right corner of each page, then clicking on the 'View Code' option.

Home Page

The home page consists of two buttons, one for the Native Components and one for the Cordova Plugins. They take you to the respective pages.

Cordova Plugins

This page contains links to the various cordova plugins displayed in the Kitchen Sink.

Native Elements

This is where the links to the various Ionic native elements are found. At the top of the page is a search box where you can filter through the displayed elements by searching for keywords eg.Searching for 'spinner' will filter for Spinner.
